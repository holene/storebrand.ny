Storebrand.ny - Version 1.2.0

Web manual and Form manual as of 8 March 2018

Notice:Versions 0.94 and upwards are WCAG-wise fine. If your application is on an older version than that, please update.

What's new in the last few versions, you ask? Well:

  • 1.2.0:
    • Bumping minor version to reflect that we're transitioning away from the old png-sprite icons.
    • Removed old png-sprite icons from the examples (still available in the css, for backwards-compatability), add link to InVision DSM which has the new icon sets
    • Buttons: all buttons now have a border-radius of 2px (previously 0)
  • 1.1.0:
    • Bootstrap vulnerability bump from 3.3.6 to 3.3.7.
    • jQuery vulnerability bump from 2.1.0 to 3.3.1 (so yes, you may need to rephrase some of your jQuery scripts)
    • Moment vulnerability bump from 2.8.3 to 2.20.1
    • jquery-match-height functionality bump from 0.7.0 to 0.7.2
    • As a result of the bumps, the scrollable tables (Zurb) are not working in this version of the Web manual. We haven't decided what to do about that yet.
    • The fund list table has been removed from the web manual, heavily hit by real-life need for significant local adjustments in almost every instance.
    • scrollTo function has been renamed stbScrollTo because of clash with native window.scrollTo.
    • New support color; stb-support-aqua, #1d8d9b.
    • YouTube player fix, so that video doesn't restart after a while when container iframe is hidden.
  • 1.0.8: Minor fix/workaround for IE bug concerning search URLs containing Norwegian characters
  • 1.0.7: Added shopping cart utility functions (PR)
  • 1.0.6: Updated Form manual with simpler and better slider.

How to play

Here's the gzipped code you'll want to include to start playing with this version:

 <link rel="stylesheet" type="text/css" href="//elements.storebrand.no/storebrand.ny/1.1.0/css/stb-all.min.css">
 <script type="text/javascript" src="//elements.storebrand.no/storebrand.ny/1.1.0/js/stb-all.min.js"></script>
